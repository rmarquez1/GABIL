GABIL
=====
Para correr el programa:
Utilice: 
	python gabil.py iris.data tasaMutacion tasaCrossover tipoSeleccion penalizacion
Donde:
	* tipoSeleccion puede tomar los valores:
		1 para Rueda de Ruleta
		2 para Otro

	* penalizacion puede tomar los valores: 
		1 sin penalización
		2 con penalización
